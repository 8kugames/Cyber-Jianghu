# Agent 规划中的功能

### 已实现

- 完整的 Review System（三层分级审核 + reject 反馈增强）
- Prompt 配置化（`prompt_templates.yaml` 模板驱动，生存/sanity warning 数据注入）
- 地图分层（parent-child 隐式连接，sub_scene 可直达 parent）
