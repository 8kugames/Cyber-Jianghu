## 三魂架构

```
人魂 ActorSoul（行动之魂）      天魂 ReflectorSoul（守护之魂）
  │  地魂: tool calling 工具池      │
  │                              │
  │  结构化 Intent               │  三层审查
  │  action_type=eat             │  action_type 合法性
  │  action_data={item_id:       │  RuleEngine 规则校验
  │    "mantou"}                 │  LLM 人设/世界观审查
  │  ──────────────────────>     │
  │                              │
  │<───── Approved: send intent ─│
  │<───── Rejected: retry ───────│
```

人魂直连 WorldState，地魂提供 tool calling 工具池（行动落地），
天魂对 Intent 进行三层审查。驳回原因通过 last_rejection_reason
跨 tick 反馈给人魂。

## 数据流

```
Server ─[WebSocket]→ infra/transport ─[WorldState]→ runtime ─[Intent]→ infra/transport ─[WebSocket]→ Server
```
