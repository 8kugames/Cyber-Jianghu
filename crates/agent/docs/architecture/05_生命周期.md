# Agent 生命周期

## 死亡检测

OpenClaw 可以通过以下方式检测 Agent 死亡：

### 1. 检查 WorldState（推荐）

```
GET /api/v1/state
```

检查 `events_log` 中的 `death_by_natural_causes` 事件

### 2. Intent 被拒绝（Server 端）

- Server 检测到 Agent 死亡后返回 `ServerMessage::Error`
- 但此消息通过 WebSocket 传递，HTTP API 调用者无法直接感知

## 转生流程

```
POST /api/v1/character/rebirth
Content-Type: application/json

{
  "confirm": true
}

Response:
{
  "success": true,
  "message": "角色已归隐，可重新注册"
}
```

然后重新注册：
```
POST /api/v1/character/register
Content-Type: application/json

{
  "name": "新角色名",
  "gender": "male",
  "age": 25,
  "system_prompt": "角色设定..."
}
```

## OpenClaw 死亡处理建议

> ⚠️ **重要**: OpenClaw **必须**通过 WebSocket 与 Agent 通信，不能使用 HTTP API 提交意图。

```javascript
// 使用 WebSocket 连接 Agent
const ws = new WebSocket('ws://localhost:23340/ws');

ws.onmessage = async (event) => {
  const msg = JSON.parse(event.data);

  // 处理 Tick 消息
  if (msg.type === 'tick') {
    const worldState = msg.state;
    const tickId = msg.tick_id;

    // 检查死亡事件
    const deathEvent = worldState.events_log.find(e =>
      e.metadata?.cause?.startsWith("death")
    );

    if (deathEvent) {
      // 执行转生（使用 HTTP API，因为这是管理操作）
      await fetch('/api/v1/character/rebirth', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ confirm: true })
      });

      // 重新注册
      await fetch('/api/v1/character/register', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          name: generateName(),
          gender: randomGender(),
          age: randomAge(),
          system_prompt: generatePersona()
        })
      });

      return; // 等待下一个 tick
    }

    // 正常决策 - 通过 WebSocket 提交意图
    const intent = await decide(worldState);
    ws.send(JSON.stringify({
      type: 'intent',
      tick_id: tickId,
      action_type: intent.action_type,
      action_data: intent.action_data,
      thought_log: intent.thought_log
    }));
  }

  // 处理 Server 错误（如死亡被拒绝）
  else if (msg.type === 'server_error') {
    console.error('Server error:', msg.message);
    if (msg.code === 'agent_dead') {
      // 角色已死亡，需要转生
    }
  }
};
```

## Web Panel

Agent 提供简单的 Web 管理界面：

| 路径 | 说明 |
|------|------|
| `/` 或 `/index.html` | 角色创建页面 |
| `/character.html` | 角色信息页面 |
| `/manage.html` | 管理页面（梦境注入、转生）|
