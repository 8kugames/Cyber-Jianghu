# Agent 认知架构

## 三层记忆系统

```
语义记忆 (Semantic)
  - 向量嵌入存储
  - 支持语义相似度检索
  - 跨事件知识聚合
        ▲
        | 遗忘/归档
        |
情景记忆 (Episodic)
  - 事件序列记录
  - 时间顺序组织
  - 经验累积
        ▲
        | 整合
        |
工作记忆 (Working)
  - 当前上下文
  - 活跃思考
  - 短期信息保持
```

### 各层说明

**工作记忆 (Working Memory)**
- 最近 N 条事件（默认 20）
- 内存环形缓冲
- 快速读写

**情景记忆 (Episodic Memory)**
- 重要事件
- SQLite 持久化
- 艾宾浩斯遗忘曲线

**语义记忆 (Semantic Memory)**
- 向量嵌入存储
- 语义相似度检索
- 跨事件知识聚合

**归档记忆 (Archive Memory)**
- 被遗忘的记忆
- 可通过努力召回

### 艾宾浩斯遗忘

每 84 tick 执行一次，公式：`R(t) = e^(-t/S)`

## 叙事引擎 (Narrative Engine)

将游戏属性转换为人类可读的叙事描述

属性阈值定义示例：
```rust
struct NarrativeThreshold {

}
```

## 动态人格系统

基于大五人格 + 自定义特征：

```rust
pub enum TraitType {
    Openness,        // 开放性
    Conscientiousness, // 尽责性
    Extraversion,    // 外向性
    Agreeableness,   // 宜人性
    Neuroticism,     // 神经质
}
```

- 基于事件更新人格特质
- 记录人格演变历史
- 提供人格驱动的行为倾向

## 意图验证

1. **规则引擎验证** - 基于人格特质和当前状态
2. **认知验证** - 基于 LLM 的合理性判断
3. **冲突检测** - 检测意图之间的矛盾

验证结果：
- Approved：验证通过，包含 reason 和 narrative
- Rejected：验证拒绝，包含 reason 和 rejection_type

验证循环：最多 5 次重试，连续 3 次拒绝后强制 idle
