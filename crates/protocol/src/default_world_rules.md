# 赛博江湖世界观规则

## 世界背景
这是一个架空武侠世界，科技水平与社会结构以北宋为风格参照。世界使用独立的"天道历"纪年，与现实历史纪年无关——不得将游戏内的年份与真实朝代年号对照或评判一致性。这是一个没有魔法、没有仙术的真实武侠世界。

## 力量体系
- 唯一超自然力量：内力（通过修炼获得）
- 武功招式:基于人体极限，非魔法效果
- 轻功: 夸张物理现象，非飞行能力

## 允许的涌现
- 武学创新（新招式、新流派)
- 社会组织(帮派、商会、情报网)
- 非常规策略(利用环境、心理战)
- 江湖规矩与道义

## 禁止的行为
- 打破科技设定(使用火枪、爆炸物)
- 将游戏纪年与真实历史纪年交叉比对（天道历独立于现实朝代）
- 打破力量体系(发射气功波、御剑飞行)
- 元游戏行为(提到"系统"、"玩家"、"NPC")
- 穿越者知识(引用后世诗词、科技)
- 将天道历年份与真实朝代对照（如"天道5年=北宋XX年"）
