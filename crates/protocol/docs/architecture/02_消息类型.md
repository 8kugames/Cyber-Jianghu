# Protocol 消息类型

## ServerMessage

Server 发送给 Agent 的消息：

```rust
pub enum ServerMessage {
    Registered {
        agent_id: Uuid,
        game_rules: GameRules,
        world_building_rules: WorldBuildingRules,
    },
    WorldState {
        tick_id: i64,
        self_state: AgentSelfState,
        location: Location,
        entities: Vec<Entity>,
        events_log: Vec<Event>,
    },
    Error {
        message: String,
    },
    Dialogue {
        from: Uuid,
        message: String,
    },
}
```

## ClientMessage

Agent 发送给 Server 的消息：

```rust
pub enum ClientMessage {
    Intent {
        intent_id: Uuid,
        tick_id: i64,
        thought_log: Option<String>,
        action_type: ActionType,
        action_data: Option<serde_json::Value>,
        priority: i32,
    },
    Dialogue {
        message: DialogueMessage,
    },
}
```

## WorldState

每个 Tick 广播的完整世界状态快照：

```rust
pub struct WorldState {
    pub tick_id: i64,
    pub self_state: AgentSelfState,
    pub location: Location,
    pub entities: Vec<Entity>,
    pub events_log: Vec<Event>,
}
```
