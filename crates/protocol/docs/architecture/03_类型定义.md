# Protocol 类型定义

## actions.rs

动作类型定义：

```rust
pub enum ActionType {
    Idle,
    Move,
    Look,
    Attack,
    Defend,
    Speak,
    Use,
    Drop,
    Pickup,
    Equip,
    Craft,
    Trade,
}
```

## attributes.rs

属性类型定义：

```rust
pub enum AttributeType {
    Primary,
    Derived,
    Status,
}

pub struct Attribute {
    pub name: String,
    pub value: i32,
    pub attr_type: AttributeType,
}
```

## entities.rs

实体类型定义：

```rust
pub struct Entity {

    pub name: String,
    pub entity_type: EntityType,
    pub position: Position,
    pub state: serde_json::Value,
}
```
