# Cyber-Jianghu Protocol 概述

## 定位

Protocol crate 定义了 Server 和 Agent 之间共享的所有类型，包括消息格式、数据模型、验证规则等。

## 模块结构

```
src/
├── lib.rs                   库入口
├── error.rs                 错误类型
├── messages.rs              消息类型（ServerMessage, ClientMessage）
├── sqlx_types.rs           SQLx 兼容类型
│
└── types/                  数据类型
    ├── mod.rs
    ├── actions.rs           动作类型
    ├── attributes.rs        属性类型
    ├── entities.rs          实体类型
    ├── locations.rs         地点类型
    ├── narrative.rs         叙事配置类型
    ├── review.rs            审核类型
    ├── rules.rs             规则类型
    └── world.rs             世界状态类型
```
