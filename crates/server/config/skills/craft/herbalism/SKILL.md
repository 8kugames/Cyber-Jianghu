# 药草学

## 何时使用
- 执行"采集"动作时
- 执行"制造"动作（配方涉及药材）时
- 接触未知植物/食物时判断是否可用

## 行为指引
- 采集时注意辨识植物特征，避免误采毒草
- 同一地点重复采集，产量递减
- 珍稀药草仅在特定地形（山林、湿地）刷新
- 制造药品需记忆或携带配方，遗忘后仅能制作基础配方
- 野外采集的药草使用前需判断是否安全
