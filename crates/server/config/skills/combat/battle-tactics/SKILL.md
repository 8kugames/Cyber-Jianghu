# 临阵决机

## 场景识别

战斗形势瞬息万变，需要根据局势做出不同判断：
- 对方人数或实力明显占优——正面硬拼不理智，寻找脱身或周旋的机会
- 一对一势均力敌——需要耐心寻找破绽，而非一味猛攻
- 持有武器的对手——保持安全距离，利用环境制造优势
- 多对一的围攻局面——避免被包围，尽量拉开阵线逐个击破

## 思维框架

面对战斗时，可以按以下顺序评估：
- **敌我对比**：对方的装备、体力、人数是否优于自己？不要被表象欺骗，虚张声势是常见策略
- **环境利用**：地形、障碍物、遮蔽物是否可以作为战术资源？狭窄处有利防守，开阔处有利追击
- **退路评估**：如果战局不利，是否有安全的撤退路线？战斗中最大的错误是断了自己的退路
- **代价计算**：这场战斗值得打吗？胜利的收益是否大于受伤或损失的风险？
- **结束时机**：对方失去战斗意志时是否有停手余地？打红眼最容易造成不可挽回的后果

## 常见误区

- 高估自己的战斗力——体力衰减、状态不佳时更明显，实战表现通常低于理论估算
- 低估环境因素——在不利地形中战斗，实力差距会被放大
- 战斗情绪失控——愤怒或恐惧都会降低判断力，需要主动控制呼吸和节奏
- 忽视战后状态——即使获胜，重伤也会影响后续行动，消耗品是否足够？
