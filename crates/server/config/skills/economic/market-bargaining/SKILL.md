# 市井权衡

## 场景识别

交易场景多种多样，需要识别不同的谈判形势：
- 日常消耗品交易——价格透明，以快速成交为主，不必过度纠缠
- 稀缺物品交易——价值难以评估，需要多方询价再做决定
- 情报或服务的交易——无形的商品更难定价，需要评估其实际效用
- 长期合作关系的建立——单次交易的价格不是最重要的，信任和稳定性更关键

## 思维框架

面对交易时，可以考虑以下原则：
- **价值锚点**：了解物品的基本价值（如市场价格、制作成本），不被对方的报价带偏判断
- **替代方案**：如果这笔交易不成，还有其他渠道获得所需吗？有替代方案时议价空间更大
- **信息不对称**：对方比自己更了解物品的真实情况吗？稀缺性、品相、来源都是信息差
- **时间窗口**：对方是否急于出手？急着买或急着卖的一方通常需要付出溢价或接受折扣
- **关系投资**：在可接受的范围内，让渡一点利益可能换来长期的合作意愿

## 常见误区

- 只看价格不看价值——便宜的东西如果用不上就是浪费，贵的东西如果解燃眉之急就值
- 过度砍价——把对方压得太狠可能破坏长期关系，下次对方不会再给你好条件
- 轻信稀缺话术——"就这一件了"、"其他人也在问"是常见的施压话术，需要验证
- 忽视隐性成本——运输、保存、风险、时间成本都是实际成本的一部分，不只看交易价格
