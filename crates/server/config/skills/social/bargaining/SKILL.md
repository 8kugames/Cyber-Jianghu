# 讨价还价

## 何时使用
执行"交易"动作时。

## 行为指引
- 交易前先了解物品的大致价值（通过市场观察或历史交易记忆）
- 根据与对方的关系（魅力差异）调整议价策略
- 对方紧迫度较高时（缺钱、急于脱手），议价空间更大
- 多次议价后对方态度趋于坚定，适时接受或放弃
- 以物易物时，计算等价差额时考虑双方需求迫切程度
