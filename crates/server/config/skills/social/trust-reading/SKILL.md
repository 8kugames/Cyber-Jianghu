# 识人之明

## 场景识别

以下情况值得主动评估对方的可信度：
- 陌生人主动接近并提出有利条件
- 对方急于达成某种协议或要求快速行动
- 对方的行为模式与之前的表态不符
- 涉及重要资源或信息的交换

## 思维框架

评估可信度时，可以考虑以下维度：
- **言行一致性**：对方之前说过的话和实际行为是否对得上？反复改变立场的人不太可靠
- **利益结构**：对方从这次互动中获得什么？如果收益明显不对等，需要警惕
- **时间压力**：对方是否催促立即决定？真正的善意通常允许对方思考
- **历史记录**：与对方有过往互动吗？之前的行为是预测未来行为的最佳依据
- **替代解释**：对方的行为是否有其他合理的解释？不要急于下结论

## 常见误区

- 把热情当真诚——过度热情可能是有目的的表演，尤其是初次见面
- 一次善意推导终身可靠——单次行为不足以建立完整信任，需要多次验证
- 认为熟人必然可信——关系亲近的人也可能因利益变化而改变态度
- 忽视不一致的信号——当直觉感到"哪里不对"时，通常确实有问题
