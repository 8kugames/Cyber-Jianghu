# 见微知著

## 场景识别

以下场景需要提高感知敏感度：
- 刚到达一个新地点——需要快速了解环境和潜在风险
- 熟悉的环境出现异常变化——物品移位、人变少、气氛紧张
- 观察到某人的行为模式突然改变——可能有重要事件发生
- 直觉感到不安——直觉往往是潜意识对环境信号的处理结果

## 思维框架

感知环境时，可以关注以下维度：
- **人物扫描**：谁在场？谁是新面孔？谁不见了？谁在注意自己？
- **环境变化**：出入口是否通畅？遮蔽物在哪里？资源分布如何？
- **行为模式**：周围人的行为是否正常？是否有人在观察自己？是否有异常的安静或躁动？
- **自身状态**：自己的体力、装备、药品是否足以应对突发情况？
- **退出路线**：如果情况恶化，最快的离开方式是什么？

## 常见误区

- 忽视直觉——大脑在注意到环境异常时会发出不安信号，不要压抑这种警告
- 只关注显性威胁——最大的危险往往来自看似无害的变化（熟人态度转变、环境微妙改变）
- 一次扫描就放心——环境是动态的，需要持续关注变化
- 过度警惕——持续的过度警觉会消耗精神和体力，需要区分真实威胁和假警报
