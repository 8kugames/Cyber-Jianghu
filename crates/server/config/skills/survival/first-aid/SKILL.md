# 急救术

## 何时使用
- 执行"使用"动作（药品）时
- 处理自身或他人伤势时
- 野外接触未知植物时

## 行为指引
- 优先处理持续失血，止血优先于其他治疗
- 使用药品注意用量和冷却时间（同种药品间隔30秒）
- 重伤（生命值低于30%）时药品效果下降
- 野外自采药草使用前需判断安全性
- 多人协作急救时效率更高
- 战斗中急救成功率下降，仅限紧急情况
