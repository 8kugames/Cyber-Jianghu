# 未雨绸缪

## 场景识别

资源管理的关键时刻：
- 刚获得大量资源（食物、药品、物品）——如何分配？
- 资源储备低于安全线——是否需要立即补充？
- 需要消耗资源的决策（战斗消耗体力、制作消耗材料）
- 可以分享或交易资源时的权衡

## 思维框架

规划资源时，可以考虑以下原则：
- **安全线思维**：为关键资源设定最低储备量（食物、药品），低于此线时补充优先级最高
- **优先级排序**：生存（食物/水）> 健康（药品）> 能力（工具/武器）> 社交（交易品）
- **消耗节奏**：资源消耗的速度决定补充的紧迫性。了解自己的日均消耗量
- **多元化储备**：不要把所有储备押在单一来源上。多种获取渠道更安全
- **时间价值**：现在的 1 份食物和 3 天后的 1 份食物价值不同——越稀缺时价值越高

## 常见误区

- 即时消费倾向——获得的资源立刻用完，不留余量应对突发情况
- 囤积癖——过度储备导致错过使用资源的最佳时机（如该治疗时不治疗）
- 忽视消耗速度——不知道自己的日均消耗量，导致突然发现资源耗尽
- 不愿分享——在信任的人之间适度分享资源可以建立互助网络，长期收益大于独占
