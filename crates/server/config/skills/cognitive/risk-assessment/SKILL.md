# 审时度势

## 场景识别

以下情况需要特别审慎地评估风险：
- 行动的后果可能不可逆（失去生命、失去重要资源、破坏重要关系）
- 信息不充分但必须做出决定
- 收益与风险严重不对称（小收益大风险，或大收益小风险都值得警惕）
- 多个选项各有优劣，没有明显最优解

## 思维框架

评估风险时，可以考虑以下因素：
- **最坏情况**：如果行动失败，最坏的结果是什么？自己能否承受？
- **概率判断**：根据已知信息，成功和失败的概率大概各是多少？不是"可能成功"而是"有多大可能"
- **可逆性**：这个决定能撤回吗？不可逆的决定需要更高的成功概率才值得冒险
- **机会成本**：选择 A 就意味着放弃 B。被放弃的选项是否同样有价值？
- **信息缺口**：当前还缺什么关键信息？能否在决定前获取？延迟决定的代价是什么？

## 常见误区

- 只看收益不看风险——高回报通常伴随高风险，没有免费的午餐
- 沉没成本陷阱——已经投入的资源不应该影响后续决策，关键是未来的投入产出
- 确定性偏误——倾向于选择"确定的小收益"而非"可能的大收益"，但长期看后者更优
- 从众心理——多数人的选择不一定正确，尤其是在信息不足时
