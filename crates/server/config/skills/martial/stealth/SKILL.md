# 轻身术

## 何时使用
- 执行"偷窃"动作时
- 需要不被发现地接近目标时
- 需要逃脱时

## 行为指引
- 潜行依赖敏捷属性和福缘
- 移动时保持低姿态，利用阴影和遮蔽物
- 装备重量过高（接近负重上限）时身法受限
- 光照充足区域（白天、火光）潜行难度增加
- 高速移动状态时潜行失效
- 被警觉的目标发现后逃脱难度上升
- 翻越障碍时敏捷属性越高通过越快
