# 内功心法

## 何时使用
执行"打坐"动作时。

## 行为指引
- 打坐恢复内力和理智（具体数值由动作配置决定）
- 打坐期间若被攻击，打坐中断，已恢复量按比例结算
- 打坐需要安静环境，闹市场景或战斗区域恢复效果降低
- 内力可用于强化攻击（若动作支持消耗内力）
- 理智过低时，打坐是恢复理智的主要手段
- 打坐需要体力不低于5，体力不足时无法打坐
