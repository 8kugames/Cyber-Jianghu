# Server 通信协议

## Agent → Server (WebSocket)

```
ws://server:23333/ws?device_id={uuid}&token={auth_token}
```

### 消息格式 (ClientMessage)

```json
// 提交意图
{
  "Intent": {
    "intent_id": "uuid",
    "tick_id": 123,
    "thought_log": "思考内容",
    "action_type": "idle",
    "action_data": {},
    "priority": 1
  }
}

// 对话消息
{
  "Dialogue": {
    "message": { ... }
  }
}
```

## Server → Agent (WebSocket)

### 消息格式 (ServerMessage)

```json
// 注册成功
{
  "Registered": {
    "agent_id": "uuid",
    "game_rules": { ... },
    "world_building_rules": { ... }
  }
}

// 世界状态广播
{
  "WorldState": {
    "tick_id": 123,
    "self_state": { ... },
    "location": { ... },
    "entities": [ ... ],
    "events_log": [ ... ]
  }
}

// 错误消息
{
  "Error": {
    "message": "错误描述"
  }
}

// 对话消息
{
  "Dialogue": {
    "from": "speaker_id",
    "message": "..."
  }
}
```

## HTTP API

| 端点 | 方法 | 说明 |
|------|------|------|
| `/health` | GET | 健康检查 |
| `/api/v1/agent/connect` | POST | 设备连接 |
| `/api/v1/agent/register` | POST | 注册 Agent |
| `/api/v1/agent/rebirth` | POST | 删除 Agent（转生） |
| `/api/dashboard/stats` | GET | 仪表板统计 |
| `/api/config` | GET | 配置列表 |
| `/api/config/{name}` | GET | 获取特定配置 |
