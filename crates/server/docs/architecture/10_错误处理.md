# 错误处理

## 错误类型

| 错误 | 触发条件 | 建议操作 |
|------|----------|----------|
| `AgentDead` | Agent 已死亡 | 调用 rebirth API |
| `RateLimitExceeded` | 请求过于频繁 | 等待后重试 |
| `TickIdMismatch` | 意图 tick 不匹配 | 使用当前 tick 重试 |
| `InvalidAction` | 动作参数无效 | 检查动作定义 |
| `InsufficientResources` | 资源不足 | 等待恢复或获取资源 |

## 错误消息格式

当前错误消息是人类可读的文本：

```
Agent 已死亡，无法执行此动作。请重新转生入世。
Intent tick_id 100 不匹配当前 tick 105。请提交当前 tick 的意图。
```
