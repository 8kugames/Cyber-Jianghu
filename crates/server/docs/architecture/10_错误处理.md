# 错误处理

## 错误码（协议层）

| 错误码 | 说明 | 建议操作 |
|--------|------|----------|
| `agent_dead` | Agent 已死亡 | 调用 rebirth API |
| `rate_limited` | 请求过于频繁 | 等待后重试 |
| `tick_mismatch` | 意图 tick 不匹配 | 使用当前 tick 重试 |
| `not_accepting` | 服务端未开始接受意图 | 等待下个 tick |
| `invalid_message` | 消息格式错误 | 检查消息格式 |
| `dialogue_failed` | 对话失败 | 检查对话状态 |
| `action_failed` | 动作执行失败 | 检查动作参数 |

## 错误消息格式

结构化错误码 + 人类可读文本：

```json
{
  "type": "error",
  "code": "tick_mismatch",
  "message": "Intent tick_id 100 不匹配当前 tick 105。请提交当前 tick 的意图。",
  "current_tick_id": 105
}
```

### 错误码常量

| 错误码 | 说明 |
|--------|------|
| `tick_mismatch` | 意图 tick 不匹配 |
| `not_accepting` | 服务端未开始接受意图 |
| `agent_dead` | Agent 已死亡 |
| `rate_limited` | 请求过于频繁 |
| `invalid_message` | 无效消息 |
| `dialogue_failed` | 对话失败 |
