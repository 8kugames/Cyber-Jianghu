# 对话系统

## 对话流程

```
Agent --> Server: DialogueRequest
Server --> NPC: DialogueRequest
NPC --> Server: DialogueResponse
Server --> Agent: DialogueMessage
```

## 对话会话

```rust
struct DialogueSession {
    id: SessionId,
    agent_id: AgentId,
    npc_id: NpcId,
    messages: Vec<DialogueMessage>,
    context: DialogueContext,
    state: DialogueState,
}
```
