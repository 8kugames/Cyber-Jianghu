# 数据驱动设计

## 配置文件

| 配置文件 | 说明 |
|----------|------|
| actions.yaml | 动作定义 |
| attributes.yaml | 属性定义 |
| items.yaml | 物品定义 |
| locations.yaml | 地点图 |
| game_rules.yaml | 核心规则 |
| time.yaml | 时间系统 |
| narrative_config.yaml | 叙事配置 |
| recipes.yaml | 配方 |
| inventory.yaml | 背包配置 |
| network.yaml | 网络配置 |
| world_building_rules.yaml | 世界构建规则 |
| display_messages.yaml | 显示消息模板 |

## 动作定义示例

```yaml
actions:
  move:
    description: "移动到指定地点"
    parameters:
      - name: destination
        type: location_id
        required: true
    effects:
      - type: location_change
        target: self
    validation:
      - attribute: stamina
        min: 10
```

## 属性定义示例

```yaml
attributes:
  primary:
    - name: strength
      base: 10
      min: 1
      max: 100
    - name: agility
      base: 10
  derived:
    - name: max_health
      formula: "base_health + strength * 2"
```

## 公式引擎

支持动态公式计算：

```rust
"base_damage + strength * 0.5 + weapon_damage"
"max_health * 0.1"
"agility > 50 ? critical_chance * 2 : critical_chance"
```
