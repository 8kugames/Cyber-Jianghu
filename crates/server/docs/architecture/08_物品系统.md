# 物品系统

## 物品类型

| 类型 | 说明 | 示例 |
|------|------|------|
| weapon | 武器 | sword, knife |
| armor | 防具 | helmet, shield |
| consumable | 消耗品 | potion, food |
| quest | 任务物品 | key, scroll |
| material | 材料 | wood, iron |

## 背包管理

```rust
struct Inventory {
    items: Vec<InventorySlot>,
    capacity: u32,
    max_stack: u32,
}

struct InventorySlot {
    item_id: ItemId,
    quantity: u32,
    metadata: ItemMetadata,
}
```
