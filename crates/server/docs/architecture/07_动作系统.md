# 动作系统

## 动作类型

| 类型 | 说明 | 示例 |
|------|------|------|
| basic | 基础动作 | idle, move, look |
| social | 社交动作 | talk, emote |
| combat | 战斗动作 | attack, defend |
| item | 物品动作 | equip, use, drop |
| crafting | 制作动作 | craft, cook |

## 动作执行器

### 基础动作执行器

```rust
async fn execute_basic(action: &Action, state: &mut WorldState) -> Result<Vec<Event>> {
    match action.type {
        "idle" => execute_idle(state),
        "move" => execute_move(state, &action.params),
        _ => Err(ActionError::UnknownAction),
    }
}
```
