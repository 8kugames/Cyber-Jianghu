# Cyber-Jianghu Server 概述

## 定位

Cyber-Jianghu Server 是游戏世界的"物理引擎"（天道），负责维护世界状态、执行 Tick 循环、处理 Agent 意图、结算动作、广播世界状态。

## 核心设计原则

### 1. 服务器权威 (Server Authoritative)

- 所有状态变更由服务器执行
- 客户端只能提交意图，不能直接修改状态
- 服务器验证所有操作，拒绝非法请求

### 2. Tick 只向前

- Tick 是游戏时间的基本单位
- 每个 Tick 持续 60 秒（可配置）
- 意图必须在当前 Tick 关闭前提交
- 过期的意图被 Server 拒收

### 3. 数据驱动

所有游戏机制通过配置文件定义（YAML 优先，JSON 回退）：

- `actions.yaml` - 动作定义
- `attributes.yaml` - 属性定义
- `items.yaml` - 物品定义
- `locations.yaml` - 地点图
- `game_rules.yaml` - 核心规则
- `time.yaml` - 时间系统
- `narrative_config.yaml` - 叙事配置
- `recipes.yaml` - 配方
- `inventory.yaml` - 背包配置
- `network.yaml` - 网络配置
- `world_building_rules.yaml` - 世界构建规则
- `display_messages.yaml` - 显示消息模板
