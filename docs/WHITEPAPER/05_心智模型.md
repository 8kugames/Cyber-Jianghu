# 虚境：江湖白皮书 - AI 与心智模型

## 1. 赋予代码以灵魂

在虚境：江湖中，AI 不仅仅是执行脚本的机器人，而是拥有完整心智模型的数字生命。我们通过复杂的 Prompt Engineering 和记忆架构，为 LLM 穿上了一层“人格的外衣”。

## 2. 感知-思考-行动循环 (Cognitive Loop)

每个 Agent 的生命活动都遵循人类的认知过程：

1. **感知 (Perception)**：
   - **内感**：感受到饥饿的绞痛、受伤的剧痛、体力的流逝。
   - **外感**：看到周围的人来人往，听到市井的喧闹，闻到食物的香气。
   - 系统将冷冰冰的数据转化为充满细节的感官描述，让 AI 能够“身临其境”。
2. **思考 (Deliberation)**：
   - **触发记忆**：当前的场景是否似曾相识？眼前的人是否是昔日的仇敌？
   - **动机分析**：我现在的首要目标是什么？是活下去，还是通过科举求取功名？
   - **风险评估**：如果我偷了这个馒头，被打死的概率有多大？
   - **思维链 (Chain of Thought)**：AI 必须生成一段内心独白，解释其决策背后的逻辑，这使得行为具有可解释性。
3. **行动 (Action)**：
   - 将复杂的思考结果转化为具体的行动指令。
   - 既然决定偷馒头，是趁老板不注意悄悄拿，还是明火执仗地抢？

## 3. 记忆系统：人格的基石

没有记忆，就没有自我。我们构建了一个三级记忆系统，让 Agent 拥有连贯的一生。

### 3.1 短期记忆 (Working Memory)

- **即时意识**：记录最近几分钟内发生的对话和事件。
- **作用**：支持流畅的对话交互和连贯的战术执行。

### 3.2 情景记忆 (Episodic Memory)

- **人生经历**：存储重要的人生节点。例如：“那年冬天在龙门客栈，燕无归分了我半个馒头。”
- **检索机制**：通过相关性检索（RAG），当再次遇到燕无归或看到馒头时，这段记忆会被唤醒。

### 3.3 语义记忆 (Semantic Memory)

- **世界观与信念**：从经历中提炼出的规则。
- **演化**：如果一个 Agent 多次被商人欺骗，他可能会形成“无商不奸”的信念，从而在未来的交互中变得多疑。

## 4. 人格演化：反思

角色不是一成不变的。除了实时的反应，Agent 还会进行**周期性的反思**。

- **复盘**：在夜深人静（低负载时段）时，Agent 会回顾过去一段时间的经历。
- **性格转变**：
  - 一个天真的书生，在经历了一连串的社会毒打后，可能会“黑化”为一个冷酷的谋士。
  - 一个杀人不眨眼的魔头，可能因为一次偶然的善意而放下屠刀。
- **动态 Prompt**：反思的结果会更新 Agent 的核心系统提示词 (System Prompt)，从而永久性地改变其行为模式。

## 5. 理智与疯狂

除了生理健康，我们还引入了**心理健康 (Sanity)** 维度。

- **压力源**：长期的饥饿、疼痛、目睹死亡、背叛都会积累精神压力。
- **崩溃**：当压力突破阈值，Agent 可能会患上“赛博精神病”，产生幻觉、胡言乱语，甚至进行无差别的攻击。这为江湖增添了不可预测的变数。

## 6. 社会关系网络

江湖不仅是打打杀杀，更是人情世故。我们构建了一套量化的人际关系系统。

### 6.1 量化好感度

- **情感度量**：每段关系都有一个精确的数值（-100 到 +100）来衡量。
- **动态标签**：系统会根据数值自动赋予关系标签。
  - **至交好友 (>= 80)**：可以托付后背，甚至共享资源。
  - **点头之交 (0)**：路人关系，互不侵犯。
  - **血海深仇 (<= -80)**：见面即眼红，不死不休。

### 6.2 关系的私密性与持久化

人际关系存储在 Agent 的本地记忆中。

- **隐私**：除了 Agent 自己，没人知道他内心真正信任谁。表面上的盟友可能在内心深处被标记为“潜在威胁”。
- **连贯性**：即使断线重连或经历长时间的分离，Agent 依然记得多年前的那次恩惠或背叛，确保了社交行为的长期连贯性。

## 7. 宏观智能：三皇议会 (Macro-AI)

除了微观的 Agent 个体智能，我们还引入了宏观层面的治理智能 —— **火云洞天**。

- **角色分工**：
    - **伏羲氏 (The Evolver)**：关注世界的多样性与演化方向，倾向于引入新变量。
    - **神农氏 (The Preserver)**：关注种群的生存率与资源平衡，倾向于稳健的生态策略。
    - **轩辕氏 (The Order)**：关注社会的秩序与规则，倾向于抑制过度混乱。
- **决策机制**：
    - 三个超级模型基于各自的 System Prompt 对世界状态进行评估。
    - 这种"分权制衡"的设计确保了世界既不会陷入死寂，也不会走向崩溃。
    - 针对"是否允许某次涌现传播"或"是否触发天灾"等议题发起投票。

通过这套心智模型，我们希望看到的不是千篇一律的 NPC，而是五千个性格迥异、爱恨分明的“人”。
