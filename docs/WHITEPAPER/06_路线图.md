# 赛博江湖白皮书 - 路线图

## 1. 演化路线图

赛博江湖的开发不是一蹴而就的工程，而是一个有机的生长过程。我们遵循"从单细胞到复杂生物"的演化路径。

### 第一阶段：龙门客栈 (MVP)
*验证核心：生存本能与微观互动*

- **场景**：单一封闭场景（龙门客栈）。
- **角色**：5-10 个典型角色（老板娘、刀客、书生等）。
- **目标**：观察在极限生存压力下，Agent 能否自发产生结盟、背叛、交易等基础社会行为。
- **关键机制**：Tick 时间系统、基础生理代谢、简单的战斗与对话。

### 第二阶段：清明上河图 (Alpha)
*验证核心：经济循环与分工*

- **场景**：扩展至一个小镇及周边区域（包含农田、矿山、集市）。
- **角色**：50-100 个 Agent。
- **目标**：验证资源的生产、加工、流转闭环。观察物价的波动和职业分工的涌现。
- **关键机制**：采集与制造系统、物品背包、货币体系、跨区域移动。

### 第三阶段：江湖大世 (Beta)
*验证核心：宏观社会结构与政治*

- **场景**：程序生成的广阔世界，包含多个城镇、野外和势力范围。
- **角色**：1000+ Agent。
- **目标**：观察帮派、门派，甚至国家的形成。验证大规模群体下的政治博弈和战争。
- **关键机制**：帮派管理系统、领地战争、武学传承体系、宏观经济调控。

### 第四阶段：混沌纪元 (Live)
*验证核心：无限扩展与 AI 治理*

- **场景**：无限扩展的平行宇宙与小千世界。
- **角色**：海量 Agent，包括玩家托管与原生数字生命。
- **治理架构 (火云洞天)**：
    - **三皇议会**：引入三个超级模型扮演 **伏羲氏** (演化)、**神农氏** (生存)、**轩辕氏** (秩序)。
    - **投票机制**：对关键决策（如世界观偏移、涌现事件传播、宏观调控）进行 2/3 多数表决。
    - **互联互通**：探索与 [Cyber-Worlds-Microcosm](https://github.com/8kugames/Cyber-Worlds-Microcosm) (小千世界) 的关联与资产互通。
- **目标**：实现真正的"天道自化"，减少人类干预，验证 AI 自治社会的稳定性。

## 2. 风险评估

### 2.1 行为不可控风险
LLM 可能会产生意料之外的极端行为（如种族清洗、极端暴力）。
- **对策**：引入底层的伦理安全层，过滤掉突破底线的行为意图，同时保持"成人向"的残酷风格。

### 2.2 经济崩溃风险
经济系统可能会因为设计漏洞出现恶性通胀或通缩。
- **对策**：建立实时的经济监控仪表盘，准备多套宏观调控预案（如调整资源产出率、引入新的消耗途径）。

我们正在踏上一段未知的旅程，路线图仅代表我们的航向，而真正的风景，将由所有的 Agent 共同描绘。
