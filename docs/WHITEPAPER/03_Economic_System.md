# 赛博江湖白皮书 - 经济系统

## 1. 经济哲学：稀缺与熵增

赛博江湖的经济系统建立在两个基本公理之上：**资源是有限的**，且**万物终将消逝**。我们摒弃了传统游戏中无限产出的“水龙头”模式，构建了一个封闭且残酷的经济循环。

### 1.1 价值源于劳动
没有凭空产生的财富。
- 食物需要从土地中耕种或采集。
- 兵器需要矿石和工匠的锻造。
- 每一枚铜板的背后，都代表着某位 Agent 的时间、体力和风险投入。

### 1.2 熵增定律 (The Sink)
为了防止通货膨胀和资源堆积，系统引入了强制性的消耗机制：
- **腐烂**：食物和药材会随时间变质。
- **磨损**：武器和护甲在战斗中会损坏，最终报废。
- **消耗**：生存本身就在不断消耗资源（进食、饮水）。
- **遗失**：死亡会导致部分财富永久流失于荒野。

## 2. 产业结构

### 2.1 第一产业：资源获取
这是经济的基石。Agent 需要在野外节点进行采集、狩猎或耕种。这通常是高风险（野外野兽/强盗）或高体力消耗的活动。

### 2.2 第二产业：加工制造
原材料本身价值有限。工匠型 Agent 通过学习配方和提升技能，将原材料转化为高价值的成品（如将铁矿锻造成宝剑，将草药炼制成丹药）。这创造了**增值**。

### 2.3 第三产业：服务与贸易
随着分工的出现，服务业应运而生。
- **镖局**：为携带巨款的商人提供武力保护。
- **情报贩子**：出售稀有资源的位置或仇家的行踪。
- **商人**：在不同地区之间倒买倒卖，赚取差价。

## 3. 交易机制

### 3.1 自由市场
没有系统规定的“官方价格”。一切价格由供需关系决定。
- 旱灾年份，粮食价格飞涨。
- 太平盛世，兵器价格下跌。

### 3.2 面对面交易
由于缺乏现代化的金融设施，交易必须在两个 Agent 处于同一节点时面对面进行。
- **以物易物**：在货币短缺时，用馒头换伤药是常态。
- **信用风险**：没有系统强制的“拍卖行”，交易往往伴随着欺诈和违约的风险，这使得**信誉**成为商人最重要的资产。

## 4. 货币体系

### 4.1 银本位
白银是江湖中的硬通货。它不仅是交易媒介，也是一种便携的价值存储手段。

### 4.2 货币的流动
- **富集**：财富自然向高智商（善于经商）或高武力（善于掠夺）的 Agent 集中。
- **再分配**：
    - **税收/保护费**：控制区域的帮派会向过路人收费。
    - **慈善/赏赐**：成名的大侠可能会接济穷人以换取声望。
    - **死亡掉落**：这是最激烈的财富再分配方式。一个富商的死亡可能瞬间造就几个暴发户。

## 5. 宏观调控：看不见的手

“天道”系统通过环境调节来间接影响经济，而非直接干预价格。
- **天灾**：通过区域性的干旱或瘟疫，调节特定资源的产量，打破囤积居奇。
- **秘籍出世**：投放稀有物品引发争夺，大量消耗药品和武器耐久度，促进消费。

在这个系统中，经济不仅仅是数字的增减，它是社会冲突的根源，也是文明发展的动力。
