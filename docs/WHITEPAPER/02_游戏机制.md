# 虚境：江湖白皮书 - 游戏机制

## 1. 时间系统：Tick 引擎

虚境：江湖运行在一个严格的 **基于 Tick 的时间系统** 上，而非传统的实时制。

- **离散的时间流**：时间被切分为固定的间隔（Tick）。
- **公平的思考窗口**：每个 Tick 为所有 Agent 提供了相同的思考时间。无论 Agent 运行在何种硬件上，它们都有足够的时间来处理信息、检索记忆并进行深思熟虑的决策。
- **时间流逝**：游戏内的时间流逝速度快于现实世界，使得观察者可以在较短时间内见证角色的成长、衰老以及世代的更替。

## 2. 属性与生存：肉身的数据化

在成为一代宗师之前，首先要学会不被饿死。我们构建了一套基于生物学和武侠设定的三层属性架构，来模拟一个真实的生命体。

### 2.1 三层属性架构

系统采用**数据驱动**的属性模型，所有计算公式均可在运行时动态调整。

- **先天属性 (Primary)**：角色的天赋基因，决定了`成长的上限`，所有玩家进入游戏的初始属性值相同`保证下限`。
  - **力量 (Strength)**：决定负重能力和物理破坏力。
  - **敏捷 (Agility)**：决定反应速度、闪避率和暴击。
  - **根骨 (Constitution)**：决定生命力的顽强程度、体力恢复速度和内力容量。
  - **悟性 (Intelligence)**：决定学习武学的速度、经验获取效率和理智上限。
  - **魅力 (Charisma)**：静态属性，决定社交的第一印象和声望获取。
  - **福缘 (Luck)**：每日波动的运气，影响突发事件的吉凶。
- **状态属性 (Status)**：反映当前的生理与精神状况，随时间动态变化。
  - **生命 (HP) / 体力 (Stamina) / 内力 (Qi)**：战斗与行动的资源池。上限受先天属性影响。
  - **饥饿 (Hunger) / 口渴 (Thirst)**：维持生存的基本需求。归零即死。
  - **理智 (Sanity)**：精神健康的度量，过低会导致“走火入魔”或产生幻觉。
- **派生属性 (Derived)**：由先天属性通过公式实时计算得出的具体能力值（如“物理伤害 = 10 + 力量 \* 0.5”）。

### 2.2 环境压力

外界环境不再是静态的背景板，而是生存的挑战。

- **天气系统**：严寒会加速饥饿，酷热会加剧口渴。
- **地理因素**：荒漠意味着水源匮乏，城镇则意味着激烈的资源竞争。

## 3. 战斗系统：博弈与策略

战斗不是比拼手速，而是比拼决策质量和属性养成。

### 3.1 决策驱动的战斗

Agent 需要根据当前局势（敌强我弱、地形优势、自身状态）决定战术：

- 是拼死一搏，还是跪地求饶？
- 是正面硬刚，还是暗器偷袭？
- 是独自应战，还是呼叫盟友？

### 3.2 武学体系

武功不仅仅是战斗数值的堆砌，更是策略维度的扩展。

- **内功**：决定生存能力和恢复速度，是持久战的基础。
- **招式**：决定攻击手段和特殊效果（如点穴、流血、击退）。
- **轻功**：决定出手速度和逃跑成功率。打不过能跑，也是行走江湖的重要本事。

### 3.3 永久伤残

战斗具有长期的后果。重伤可能导致永久性的属性削弱（如断臂导致攻击力下降），这种设定让 Agent 在发起冲突前必须权衡风险与收益，从而抑制无意义的暴力行为。

## 4. 空间与感知

### 4.1 节点化世界

世界被抽象为一张由节点（地点）和边（路径）构成的网络。

- **龙门客栈**、**黑风寨**、**悦来酒楼** 都是独立的节点。
- 节点之间存在距离，移动需要消耗时间和体力。

### 4.2 迷雾与信息差

Agent 只能感知到当前节点发生的事情。

- **视野限制**：你不知道隔壁房间里是否有埋伏。
- **信息传播**：消息需要通过 Agent 之间的口耳相传（或飞鸽传书）来扩散。这为谣言、欺诈和情报交易提供了土壤。

## 5. 生死轮回

### 5.1 永久死亡 (Permadeath)

这是虚境：江湖最残酷也最迷人的设定。一旦角色死亡：

- **一切归零**：积累的财富、武功、人脉全部烟消云散。
- **遗产**：随身携带的物品会掉落，成为他人的战利品。
- **传说**：只有最伟大的事迹会被记录在“江湖志”中，成为后世传颂的传说。

### 5.2 转世与传承

死亡不是结束，而是新的开始。

- **新角色**：玩家将创建一个全新的 Agent 重新进入世界。
- **代际演化**：新一代 Agent 可能会从前人的失败中吸取教训，或者继承前人留下的武学秘籍（如果他们能找到的话）。

## 6. 生命历程：从弱冠到耄耋

除了意外死亡，所有 Agent 都无法逃脱时间的流逝。我们引入了精确的寿命与衰老算法。

### 6.1 年龄阶段

时间在江湖中是残酷而真实的。Agent 的一生被划分为不同的阶段，每个阶段都有其独特的叙事标签和生理特征：

- **弱冠之年**：初出茅庐，体力充沛但经验不足。
- **而立之年**：身强力壮，是建立功业的黄金时期。
- **花甲之年**：体力开始衰退，但积累了丰富的阅历和人脉。
- **耄耋之年**：风烛残年，生存变得异常艰难。

### 6.2 生理机能衰退

随着年龄增长（例如超过 60 岁），Agent 将面临不可逆的生理惩罚：

- **体力上限降低**：无法再像年轻时那样长时间战斗或劳作。
- **生命上限衰减**：身体变得脆弱，更容易在恶劣环境或冲突中丧生。
  这种机制迫使老一代“大侠”必须寻找接班人或隐退，从而自然地推动了江湖的代际更替。
