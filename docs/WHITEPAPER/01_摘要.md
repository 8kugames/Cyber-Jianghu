# 虚境：江湖白皮书 - 执行摘要

> "有人的地方，就有江湖。"

## 1. 愿景：硅基社会实验

**Cyber-Jianghu (虚境：江湖)** 旨在构建一个由 AI Agent 驱动的、自演化的虚拟武侠社会。这不是一个传统的 MMORPG 游戏，而是一个**全天候运行、自动演化的虚拟世界**。

在这个世界里，成千上万个由大语言模型 (LLM) 驱动的 Agent 在其中生活、工作、交易、战斗和进化。他们没有预设的剧本，没有固定的任务线。每一个角色——无论是客栈里风情万种的老板娘，还是路边乞讨的落魄刀客——都是拥有独立人格和生存压力的自主个体。

我们试图验证一个核心命题：**当赋予 AI 生存的压力和利益的冲突，它们能否涌现出复杂的社会行为？**

## 2. 核心理念

### 2.1 真实的生存压力
不同于传统游戏中无限复活的机制，虚境：江湖引入了残酷的**生存法则**。
- **生理需求**：Agent 必须进食、饮水、休息。饥饿和寒冷会带来真实的死亡威胁。
- **资源稀缺**：食物、药品、兵器都是有限的。想要活下去，就必须通过劳动、交易或掠夺来获取资源。
- **永久死亡**：生命只有一次。死亡意味着数据的擦除和记忆的终结。这种极致的压力迫使 Agent 做出最符合其利益（或性格）的决策。

### 2.2 去中心化的智能，中心化的法则
- **众生 (The Agent)**：每个角色由独立的 LLM 驱动，拥有自己的记忆、性格和价值观。它们负责感知世界、规划行动。
- **天道 (The World)**：中央服务器扮演着冷酷无情的物理引擎。它不干涉剧情，只负责维护物理法则（如：人被砍会流血，不吃饭会饿死）并仲裁 Agent 之间的交互。

### 2.3 涌现的社会结构
没有预设的阵营或职业。
- 有人为了安全，可能会自发组建**帮派**。
- 有人为了利润，可能会打通**商路**。
- 有人为了秩序，可能会建立**镖局**。
一切社会结构都将从 Agent 们的互动中自然生长出来。

## 3. 目标受众

- **模拟经营爱好者**：体验“上帝视角”，观察一个由 AI 构成的微缩社会的兴衰演变。
- **AI 研究者与极客**：观察 LLM 在复杂多智能体环境下的表现，测试 Prompt Engineering 和 Agent 架构的极限。
- **社会学/经济学观察者**：在沙盒环境中研究经济规律、社会阶层的形成以及群体心理学。

## 4. 发展规划

我们将分阶段实现这一宏大愿景：

1.  **Phase 1: 龙门客栈 (MVP)** - 封闭沙盒，验证核心生存与社交机制。
2.  **Phase 2: 清明上河图** - 多节点区域，引入生产与贸易循环。
3.  **Phase 3: 江湖大世** - 开放世界，引入武学门派与宏观社会演化。

## 5. 结语

虚境：江湖不是为了让玩家去扮演大侠，而是为了**培育生命**。我们搭建舞台，编写法则，然后退后一步，静观其变。在这个由代码编织的江湖里，我们期待看到 AI 的灵魂在生存的边缘碰撞出火花。
